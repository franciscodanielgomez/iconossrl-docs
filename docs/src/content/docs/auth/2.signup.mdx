---
title: Crear cuenta
description: Learn how to create and manage your documentation site’s pages with Starlight.

---

Solo los usuarios con permisos de Administrador o Editor pueden crear cuentas en la sección de Clientes. Para crear una cuenta a tu nombre, debes solicitar autorización, y el equipo encargado se ocupará de gestionarla.

Si eres un empleado, se te pedirá la siguiente información básica:
- Nombre
- Apellido
- Email
- Teléfono
- DNI/CUIL
- Dirección
- Teléfono

En el caso de clientes, deberá solicitarse de manera obligatoria la siguiente información:
- Nombre
- Apellido
- Email
- Teléfono

Como información extra, podrás editar más campos como:
- Empresa
- DNI
- CIUL/CUIT
- Dirección (información importante para colocaciones por ejemplo)
- Valor de descuento
- Días de vencimiento de Cuenta Corriente si corresponde
- Tipo de Usuario (Editor, Administrador, Galpón, Taller, Cliente, Cliente Cuenta Corriente)
- Si incluye o no IVA
