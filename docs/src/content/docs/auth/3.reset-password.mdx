Para recuperar la constraseña debes ingresar a cualquiera de los links de las sucursales:

Sucursal Independencia: https://iconosindependencia.flutterflow.app.

Sucursal Chacabuco: https://iconoschacabuco.flutterflow.app.

Sucursal Ecommerce: https://iconosecommerce.flutterflow.app.

Una vez dentro, en el formulario de ingreso, encontrarás un enlace que dice: “Recuperar contraseña”.

Al seleccionar dicho enlace, se te pedirá ingresar tu correo electrónico registrado. Luego, recibirás un email con un enlace que deberás presionar, el cual te llevará al navegador para que puedas restablecer tu contraseña.
