[Iconos SRL SOFT ](https://iconossrl.com.ar) es una solución integral para la gestión de pedidos, diseñada para optimizar y facilitar el proceso de administración de ventas a nivel general. Este software ha sido desarrollado por [Allianz Inc](https://allianzy.com), una empresa de desarrollo de herramientas tecnológicas.

Esta guía completa está destinada a asistirte en el uso del software, proporcionando respuestas detalladas a todas las preguntas que puedas tener mientras te familiarizas con sus funciones y características. Aquí encontrarás instrucciones paso a paso, consejos prácticos y soluciones a problemas comunes, para que puedas aprovechar al máximo todas las capacidades de Iconos SRL SOFT.

Si en algún momento necesitas ayuda adicional, no dudes en consultar la documentación en línea, contactar con nuestro soporte técnico o explorar los recursos disponibles en nuestra plataforma. Estamos aquí para asegurarnos de que tu experiencia con el software sea fluida y productiva.


## Inicio Rápido


### Cómo acceder

Para acceder al sistema debes ingresar al link correspondiente a cada sucursal:

- **Sucursal Independencia:**  [https://iconosindependencia.flutterflow.app](https://iconosindependencia.flutterflow.app).

- **Sucursal Chacabuco:** [https://iconoschacabuco.flutterflow.app](https://iconoschacabuco.flutterflow.app).

- **Sucursal Ecommerce:** [https://iconosecommerce.flutterflow.app](https://iconosecommerce.flutterflow.app).


### Ingresa a tu cuenta de administrador

Si eres el propietario ya tendrás un usuario asigando, con el cual podrás acceder al sistema para dar de alta a equipo, los cuales tienen roles asignados para cada tipo de usuario.


### Tipos de roles

- **[Administrador](/guides/customization/):** Tiene el control casi absoluto en todos los elementos del sistema.
- **[Editor](/guides/customization/):** Puede realizar pedidos, dar de alta a nuevos clientes, crear cajas, y mucho más.
- **[Taller](/guides/customization/):** Solo administra los trabajos asigandos al area de Taller.
- **[Galpón](/guides/customization/):** Solo administra los trabajos asigandos al area de Galpón.
- **[Librería](/guides/customization/):** Es un editor con acceso a la edición de pro
